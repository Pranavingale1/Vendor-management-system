Django vendor management system has been created here

for testing APIs one can use thunder client in vscode or postman app

Token based authentication has been used here in which token is automatically generated with every new user

for testing APIs in thunder client one need to go to Header -->  Http header -->(Third column) Type Authorization and the token
and the change get/Post as required given in the URL section

A test case has been provided 

for a user one can create user by typing in terminal - py manage.py createsuperuser

create a virtual environment with given requirements in requirements.txt
